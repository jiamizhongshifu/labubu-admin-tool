# Canvas API 说明文档

## 什么是Canvas API？

Canvas API是HTML5的标准API，由所有现代浏览器原生支持。它允许在网页上绘制图形、处理图像和进行像素级操作。

## 费用说明

**Canvas API完全免费！**

- ✅ **无需API密钥**：Canvas是浏览器内置功能
- ✅ **无需付费订阅**：不涉及任何外部服务
- ✅ **无使用限制**：可以无限制使用
- ✅ **隐私保护**：所有处理都在本地浏览器中完成
